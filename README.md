# Run and deploy the AssessMax Plus application

This contains everything you need to run the app locally.

View your app in the deployment console: https://your-deployment-console.example.com

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your API key
3. Run the app:
   `npm run dev`
